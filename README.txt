Project Title: Electricity Bill Calculator

Description:
This program calculates electricity bills based on units consumed
and rate per unit. It demonstrates function overloading through
multiple constructors and setData functions. It also demonstrates
operator overloading through +, -, *, <, ==, and << operators.
The bill is calculated using a slab system where higher units
are charged at higher rates.

Files Included:
- main.cpp
- ElectricityBill.h
- ElectricityBill.cpp
- electricitybillcalculater.slnx

How to Run:
1. Open electricitybillcalculater.slnx in Visual Studio
2. Build the project
3. Run the program
4. Enter units and rate as prompted
